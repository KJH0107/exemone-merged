SECTION 01

프로젝트 개요

exemONE의 모든 기능을 클릭 한 번으로 배울 수 있는 인터랙티브 가이드 —
서버 없이 HTML 파일 하나로 누구에게나 즉시 배포 가능

배경 및 문제

AS-IS — 현재 문제
  • 신입 온보딩 시 담당자 동행 2~3시간 필요
  • PDF·캡처 자료로는 화면 흐름 파악 불가
  • 외부 발표 시 실서버 접속·인터넷 의존
  • 설명이 사람에게 집중되어 재현 불가
TO-BE — 개선 방향
  • 자기주도 학습으로 40~60분 내 기능 파악
  • 실제 UI에서 클릭·필터·탭 전환 직접 체험
  • 단일 HTML 파일로 인터넷 없이 어디서든 시연
  • 퀴즈 + 실습으로 기억 정착률 향상

활용 대상

대상활용 방식기대 효과
신입 구성원 입사 초기 온보딩 자기주도 학습 담당자 없이 독립적 기능 습득
외부 고객 / 발표 HTML 파일로 오프라인 인터랙티브 데모 실서버 의존 제거, 안정적 시연
기존 구성원 신규 기능 업데이트 자체 학습 별도 교육 없이 빠른 기능 파악

3
직접 해보기 (행동 확인) "상단 빨간색 Critical 카드를 직접 클릭해보세요" → 화면에서 완료 감지 → 성공 메시지
