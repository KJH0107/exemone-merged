현재 개발 중인 exemONE 인터랙티브 기능 가이드 앱에 대해 보고드립니다.


■ 프로젝트 목적

exemONE 플랫폼의 다양한 기능을 신입 구성원과 외부 고객에게
효과적으로 전달하기 위한 인터랙티브 가이드 앱입니다.

기존에는 담당자가 직접 2~3시간 설명하거나 PDF 자료를 활용하였으나,
화면 흐름 전달에 한계가 있었습니다.

본 앱은 실제 exemONE UI와 동일한 화면에서 클릭 한 번으로 기능 설명,
퀴즈, 직접 실습까지 이어지는 자기주도형 학습 환경을 제공합니다.


■ 핵심 기능

1. 우측 가이드 패널 (360px)
   - UI 요소 클릭 시 해당 기능 설명 즉시 표시
   - 기능 설명 → 퀴즈 → 직접 해보기 3단계 구성

2. 장애 분석 플로우 가이드
   - 홈 화면 → 데이터베이스 오버뷰 → 인스턴스 상세(8탭)까지
     실제 장애 대응 흐름을 단계별로 체험

3. 단일 HTML 파일 배포 (개발 목표)
   - 서버 없이 파일 하나로 어디서든 실행 가능
   - 외부 발표 시 인터넷 환경 불필요


■ 현재 진행 상황

완료 : 데이터베이스 오버뷰 + 인스턴스 상세 8탭 가이드
진행 : 홈 / 성능 분석 / 알람 페이지 가이드 연결
예정 : 단일 HTML 변환 → 신입 교육 / 외부 발표 적용


■ 기대 효과

- 신입 온보딩 시간: 2~3시간 → 40~60분 (자기주도 학습)
- 외부 데모: 실서버 의존 → 단일 파일로 즉시 시연
- 기능 이해도: 구두 설명 대비 퀴즈+실습으로 기억 정착률 향상
