## 한 줄 요약

> **exemONE의 모든 기능을 클릭 한 번으로 배울 수 있는 인터랙티브 가이드 — 서버 없이 HTML 파일 하나로 배포.**

---

## 1. 왜 만드나요?

### 현재 문제

```
신입 입사
    │
    ▼
담당자와 함께 exemONE 화면 설명 듣기 (2~3시간)
    │
    ▼
실서버에서 직접 써보며 익히기
    │
    ▼
기능 반만 파악한 채로 실무 투입
```

- 담당자가 붙어서 설명해야 해 **온보딩 비용이 사람에게 집중**됨
- 외부 발표 시 실서버 접속이 필요해 **네트워크 환경에 따라 데모 실패 위험**
- 기능이 많고 화면이 복잡해 PDF/캡처 자료로는 **흐름 파악 불가**

```
│  ┌─────────────────────────────┐  ┌─────────────────┐  │
│  │                             │  │  가이드 패널    │  │
│  │   데이터베이스 오버뷰       │  │  ─────────────  │  │
│  │                             │  │  Instance Card  │  │
│  │  [ Active 12 ] [ Critical 3]│◄─┤  ─────────────  │  │
│  │                             │  │  카드 클릭 시   │  │
│  │  ┌──────┐ ┌──────┐ ┌──────┐│  │  해당 상태만    │  │
│  │  │  DB1 │ │  DB2 │ │  DB3 ││  │  필터링됩니다   │  │
```

## 3. 학습 경험 흐름

### 퀴즈 + 행동 확인 챌린지

```
① 기능 클릭
      │
      ▼
② 설명 읽기
   "카드 클릭 시 해당 상태 인스턴스만 필터링됩니다"
      │
      ▼
③ 객관식 퀴즈
   Q. Critical 카드를 클릭하면?
      A) 인스턴스가 재시작됩니다
    ▶ B) Critical 인스턴스만 목록에 필터링됩니다  ← 정답
      C) 알람이 해제됩니다
      D) 상세 설정 화면이 열립니다
      │
      ▼ 정답 선택 시 즉시 피드백
④ 직접 해보기
   "상단 빨간색 Critical 카드를 직접 클릭해보세요"
      │
      ▼
⑤ 완료 감지 → "정확합니다! 이렇게 범위를 즉각 좁힐 수 있습니다"
```

## 4. 장애 분석 플로우 가이드

실무에서 가장 중요한 흐름인 **장애 인지 → 원인 파악**을 단계별로 체험합니다.

```
홈 화면
  └─ Critical 인스턴스 발견
        │
        ▼
데이터베이스 오버뷰
  └─ Instance Card 클릭 → Critical만 필터링
        │
        ▼
인스턴스 상세 슬라이드 (8개 탭)
  ├─ [세션] 액티브 세션 급증 확인
  ├─ [SQL]  슬로우 쿼리 목록 확인
  ├─ [Lock] 락 경합 여부 확인
  └─ [호스트] CPU/Memory 현황 확인
        │
        ▼
원인 파악 완료
```

> 가이드 패널이 각 단계마다 "지금 여기서 무엇을 봐야 하는가"를 실시간 안내합니다.

## 5. 커버 기능 현황

### 페이지별 구성

| 페이지 | 포함 기능 | 상태 |
|---|---|---|
| 홈 | 상태 요약 카드 / 위젯 커스텀 / 최근 대시보드 / 실시간 알람 / 장애 인스턴스 목록 | 콘텐츠 완료, 화면 연결 예정 |
| 데이터베이스 오버뷰 | Instance Card / 인스턴스 맵 / 필터 / 인스턴스 목록 | **완료** |
| 인스턴스 상세 (8탭) | 세션 / SQL / Lock / Replication / Vacuum / Event / Parameter / Host Process | **완료** |
| 성능 분석 | Scatter / Trend 분석 | 기획 중 |
| 알람 | 알람 현황 / 알람 규칙 / 알람 정보 | 기획 중 |

```
│                                                         │
│  액티브 세션 목록                    ┌───────────────┐  │
│  ┌──────────────────────────────┐   │  가이드 패널  │  │
│  │ PID  │ User │ State │ Query  │   │  세션 탭 선택 │  │
│  │ 1234 │ app  │ active│ SELECT │   │  시 자동 동기 │  │
│  │ 5678 │ db   │ idle  │ -      │   │  화됩니다     │  │

│                                                         │
└─────────────────────────────────────────────────────────┘
       ↑ 탭 클릭 시 가이드 패널 자동 전환 (양방향 동기화)
```

## 6. 배포 목표 — 단일 HTML 파일

### 현재 vs 목표

| 구분 | 현재 (개발 서버) | 목표 (단일 HTML) |
|---|---|---|
| 실행 방법 | `npm run dev` 후 localhost:6063 접속 | 파일 더블클릭 |
| 공유 방법 | 서버 환경 필요 | 이메일 첨부 / USB |
| 오프라인 | 불가 | 가능 |
| 외부 발표 | 인터넷 필요 | 인터넷 불필요 |

```
[개발 완료]
    │
    ▼
npm run build → 정적 파일 추출
    │
    ▼
CSS + JS + 이미지 → 단일 index.html 인라인
    │
    ▼
파일 하나로 전달 → 어디서든 실행
```
